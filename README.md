# Gamerwikihub 🎮

**Gamerwikihub** is a lightweight, all-in-one desktop application designed for gamers who want quick, uninterrupted access to their favorite game wikis, guides, and interactive maps without the browser bloat.

## ✨ Features

*   **All-in-One Gaming Wiki Hub:** Quick access to `Wiki.gg`, `Fandom`, `Stardew Valley Wiki`, `Arc Raiders Wiki`, and `Fextralife`.
*   **Fandom No-Bloat:** Built-in optimization to strip away annoying ads, videos, and pop-ups from Fandom pages for a clean reading experience.
*   **Genuimaps Integration:** Seamless access to Geniymaps with premium maps unlocked for better navigation and tracking.
*   **Lightweight & Fast:** Low resource consumption compared to opening multiple browser tabs.
*   **Built in Adblocker** Experience zero ads while navigating.
*   **always on top** So you can pin your wiki while gaming

## 🚀 How to Download & Install

1. Go to the [Releases](https://github.com/Latif-teto/GamerWikiHub/releases) section on the right side of this page.
2. Download the latest `Gamerwikihub-Installer.exe`.
3. Run the installer and follow the setups.
4. Enjoy a bloat-free gaming wiki experience!
